# my.ai

Personal workspace for conversations with Claude Code in VS Code. This repo is a private, local-first space for work that doesn't belong in web-based AI chat interfaces (claude.ai, ChatGPT, Gemini, etc.), giving full control over conversation history, files, and outputs without relying on third-party account storage.

## What This Repo Is Used For

- Creative writing, brainstorming, and drafting
- Personal projects, ideas, and explorations that benefit from AI collaboration
- Private conversations and work that should stay local
- Resume and career development
- Research and structured analysis

## Repo Structure

| File / Folder | Purpose |
|---|---|
| `CLAUDE.md` | Instructions for Claude Code sessions: required reading, writing style guidelines, and reference file locations |
| `personalwiki.md` | Comprehensive personal and professional wiki covering background, expertise, projects, interests, and preferences |
| `personality.md` | Personality and tone definitions for coaching and accountability conversations |
| `resume/` | Current resume versions and Microsoft Certification Transcript |

## How It Works

Every new Claude Code session in this repo reads `CLAUDE.md` on startup, which directs it to read the personal wiki and personality files for full context. Writing style guidelines and behavioral rules are enforced across all output. The `resume/` folder provides source-of-truth documents for any career or certification work.
